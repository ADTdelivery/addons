# ADT Mantenimiento Mecánico

## Sistema de Gestión de Órdenes de Mantenimiento
**Especializado en Motocicletas y Mototaxis**

---

## 📋 Descripción General

Módulo integral de Odoo para la gestión completa de talleres especializados en mantenimiento y reparación de motocicletas y mototaxis, cubriendo desde el ingreso del vehículo hasta la entrega final, con control financiero, de calidad y seguimiento.

---

## 🎯 Objetivos del Sistema

- ✅ Digitalizar el proceso completo de mantenimiento
- ✅ Garantizar trazabilidad técnica total
- ✅ Control financiero integrado
- ✅ Control de inventario y repuestos
- ✅ Gestión de garantías
- ✅ Seguimiento de mantenimiento periódico
- ✅ Validación documental y contractual
- ✅ Protección legal para taller y cliente

## 📦 Instalación

### Requisitos
- Odoo 16, 17 o 18
- Módulo `base` instalado
- Módulo `fleet` (recomendado)

### Pasos de Instalación
1. Copiar módulo a la carpeta addons de Odoo
2. Actualizar lista de aplicaciones
3. Buscar "ADT Mantenimiento Mecánico"
4. Instalar
