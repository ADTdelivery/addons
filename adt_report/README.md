ADT Report
==========

Module scaffold for ADT reporting features.

How to use
----------

1. Place this module under your Odoo addons path.
2. Update app list in Odoo and install the module.

Files
-----
- models/adt_report.py: example model
- views/adt_report_views.xml: example tree/form view
- security/ir.model.access.csv: basic access control
